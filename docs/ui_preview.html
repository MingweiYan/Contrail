Contrail / UI Direction Preview

更现代、有科技感，但不过分花哨的 3 套视觉方向

这份预览只展示视觉方向，不改业务交互逻辑。重点优化启动页、首页卡片层次、统计页数据面板感，以及“我的”页面的信息结构。 风格都尽量保持克制，避免高饱和霓虹和过多装饰。

冷静科技蓝

用深海蓝和青色做低饱和点缀，强化数据产品感。适合 Contrail 这种“记录痕迹”的定位，专业、轻盈、耐看。
